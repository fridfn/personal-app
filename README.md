# 🎧 My Portfolio Website

Sebuah website portfolio pribadi yang menampilkan siapa saya, project yang pernah saya buat, lengkap dengan seluruh informasi mengenai diri saya.

## 🚀 Fitur Utama

- Tampilan modern dan smooth dengan animasi custom
- Formulir kontak yang dapat mengirim email secara real-time
- Responsive layout (dalam pengembangan)
- Desain konsisten dengan warna dan tipografi yang seragam

## 🛠️ Teknologi yang Digunakan

- HTML, CSS Native (tanpa framework)
- JavaScript (ReactJS)
- Library animasi: ReactBits
- Pengiriman email: EmailJS
- Hosting: Vercel
